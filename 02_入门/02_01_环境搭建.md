

- 安装python3
    - 安装方法略，我是之前安装的，版本是3.6.5
- 安装Virtualenv
    - `sudo pip3 install virtualenv`
- 使用virtualenv
    - 新建文件夹myproject : `mkdir myproject`、 `cd myproject`
    - 创建虚拟环境 : `virtualenv venv -p python3`
    - 激活虚拟环境 : `source venv/bin/activate`
    - 退出虚拟环境 : `deactivate`
- 安装Django
    - `pip install django==1.11.4`
